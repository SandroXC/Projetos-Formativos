# API Hamburgueria da Blue

API desenvolvida utilizando Nest.js.

## Como fazer o clone

```bash
git clone git@github.com:SandroXC/blue-nest.git
```

## Como instalar as dependências

```bash
npm install
```

## Como rodar o projeto

### Produção

```bash
npm run start
```

### Desenvolvimento

```bash
npm run start:dev
```
